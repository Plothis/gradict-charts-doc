
# 柱状图 - Bar Chart


<section id="overview-graph-detail-content">

![chart-overview](//i.ibb.co/XbJ2ndv/9f39bc5bbcfe804eb645cb70806ebaeb.png)

柱状图，是一种使用矩形条，对不同类别进行数值比较的统计图表。最基础的柱形图，需要一个分类变量和一个数值变量。在柱状图上，分类变量的每个实体都被表示为一个矩形（通俗讲即为“柱子”），而数值则决定了柱子的高度。

与“柱状图”相近的概念还有“条形图”，二者在实际使用中可能产生不同理解。例如，在AntV的分类中，条形图是柱状图的子集，专指“横向的柱状图”。而在Excel2016的图表功能中，柱状图专指纵向、条形图专指横向。

作为人们最常用的图表之一，柱状图也衍生出多种多样的图表形式。例如，将多个并列的类别聚类、形成一组，再在组与组之间进行比较，这种图表叫做“分组柱状图”或“簇状柱形图”。将类别拆分称多个子类别，形成“[堆叠柱状图](</gradict-charts-doc/charts/stacked_column_chart>)”。再如将柱形图与[折线图](</gradict-charts-doc/charts/line_chart>)结合起来，共同绘制在一张图上，俗称“[双轴图](<#usage-scene-graph-detail>)”，等等。

不过，早已“飞入寻常百姓家”的柱状图，仍存在被误用的情况。一个典型代表就是[篡改y轴](<#unappropriate-scene-graph-detail-content>)，对读者视觉造成误导。在使用柱状图时，务必要使原点位于0。此外，注意对柱状图进行排序。依据可视化的目的、以及想突出的重点信息，确定合理的排序标准，避免柱状图看起来杂乱无章。

### 适用场景

<section id="appropriate-scene-graph-detail">

柱状图最适合对分类的数据进行比较。尤其是当数值比较接近时，由于人眼对于高度的感知优于其他视觉元素（如面积、角度等），因此，使用柱状图更加合适。

![null](//i.ibb.co/C9PxPRY/1548843959370-e81a6eba-88d1-4f07-8a69-8625ff270700.jpg?x-oss-process=image/resize,w_880)



</section>

### 不适用场景

<section id="unappropriate-scene-graph-detail">

（1）柱状图要求至少一个分类变量，它们之间是离散的（如一班、二班、三班），绘制为柱状图时，柱子与柱子之间有间隔。如果是连续型变量（如班上同学的成绩），则应当使用[直方图](</gradict-charts-doc/charts/histogram>)，绘制出每个区间的数值（如落在每个分数段的人数），柱子之间是连续的、没有间隔（有时为了美观会留出间隔，但间隔极小）。

![null](//i.ibb.co/BCBvdS8/1549176359593-40cea5b4-8b43-4607-a60a-70bc154de67c.jpg?x-oss-process=image/resize,w_860)

（2）柱状图最核心的功能是比较，比较的核心是高度。如果人为的改变高度，那么数据间的比例关系会失常。比如下图，y轴的起始被设置为90，此图中，值为107的蓝色柱子似乎有cheng se柱子（值为93）的5倍。因此，在使用柱状图时，要注意y轴的取值，从0开始。

![null](<//i.ibb.co/58BRKXj/1549176967033-98504ab5-63b2-4af9-9484-ab96bb23fdaf.jpg?x-oss-process=image/resize,w_860>)

有时候，这种高度上的“人为操作”，甚至会采用更隐蔽的形式。智利前总统Sebastian Piñera曾在竞选时展示自己的政绩，他执政的2010年有着最高的红色柱子，对手执政的2013年则矮的可怜。乍看之下颇有说服力，但仔细比对，几个柱子之间的数量关系明显失调（比如，2013与2016年相差4.5%，高度差却与蓝色的柱子，即22.8%差不多大；2010与2016年相差3.4%，高度差却远小于2013与2016年的4.5%）。显然，这张图也是动过手脚的。

![null](<//i.ibb.co/zxYn9CJ/1547625190962-5187573e-e703-41d6-8a78-a78b0e06f18c.jpg?x-oss-process=image/resize,w_906>)



</section>

## 使用场景

<div id="usage-scene-graph-detail"></div>

（1）用于分类数据的比较。下图（梅昕, 孙丹阳, 2018

）展示了海底捞公司外卖收入的变化，也是一幅“双轴图”。柱状图表示外卖收入的金额，从2015年的7500万左右上涨到了2017年的2亿余元。折线则代表了外卖收入在总收入中的占比，2015年该比例不到2%，2016年涨幅较大，2017年则相对缓慢。

![null](//i.ibb.co/6w1FHX5/1547966414396-7c3f78e4-b7f2-456a-a3f8-068ee3f3618d.png?x-oss-process=image/resize,w_768)

（2）用柱状图做时间序列分析，记录历时、历年的变化情况，这也是近年比较热门的应用场景。比如这一案例（Blackburn, 2018

），将柱状图折成一个圆圈，清晰美观的反映了1400-2013年陨石降落的情况。[[点击访问原作品网页]](<https://public.tableau.com/en-us/s/gallery/meteorite-landings?gallery=votd>)

![null](<//i.ibb.co/R2qJGzf/1547626922702-1ac99e16-86cd-4288-99fd-fce7fd41277c.jpg?x-oss-process=image/resize,w_904>)

（3）柱状图+m&m豆，用于儿童智力教育（Mrs. Thomas' Teachable Moments, 2015

）。[[点击查看原作品网页]](<https://www.pinterest.com/pin/75576099973505021/>)

![null](<//i.ibb.co/WPhkPFJ/1547569400929-e272fa9f-a4e2-4d01-b80e-b4714c3da1aa.jpg>)

（4）柱状图应用于现场投票，一种游戏化的尝试（Kimbell, 2016

）。[[点击访问原作品网页]](<http://www.lucykimbell.com/LucyKimbell/PhysicalBarCharts.html>)
