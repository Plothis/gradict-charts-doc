
# 南丁格尔玫瑰图 - Nightingale Rose Chart


<section id="overview-graph-detail-content">

![chart-overview](//i.ibb.co/dQ9Qzqt/9c82d980aa7a4ed22a3a6dabc2a20077.png)

南丁格尔玫瑰图又名鸡冠花图、极坐标区域图，统计学家和医学改革家[佛罗伦萨‧南丁格尔](<https://en.wikipedia.org/wiki/Florence_Nightingale>) (Florence Nightingale) 在克里米亚战争期间创造了这种图表，用以反映军医院的季节性死亡率（见[场景案例1](<#usage-scene-graph-detail>)），促进了医院条件的改良。

尽管外形很像[饼图](</charts/pie_chart>)，但本质上来说，南丁格尔玫瑰图更像在极坐标下绘制的[柱状图](</charts/column_chart>)或[堆叠柱状图](</charts/stacked_column_chart>)。只不过，它用半径来反映数值（而饼图是以扇形的弧度来表示数据的）。但是，由于半径和面积之间是平方的关系，视觉上，南丁格尔玫瑰图会将数据的比例夸大。因此，当我们追求数据的准确性时，玫瑰图不一定是个好的选择。但反过来说，当我们需要对比非常相近的数值时，适当的夸大会有助于分辨。

### 不适用场景

<section id="unappropriate-scene-graph-detail">

（1）分类过少的场景。例如，只有2类时，应该直接用[饼图](</charts/pie_chart>)或者环图来表示。

![null](//i.ibb.co/L6Sw0W3/1551510695062-ee8a1c1c-5c98-4d45-944a-994d74355dff.png)

（2）南丁格尔玫瑰图是将数值映射到半径上，而扇形的面积和半径是平方关系，因此视觉上看，数值的差异会被扩大。因此，当数值差异较大、或者希望精确的比较数值大小时，推荐使用[柱状图](</charts/column_chart>)。

![null](<//i.ibb.co/Qdv7hZx/1551510682034-2c042d73-5598-469f-b272-7dee9252de21.png>)



</section>

## 使用场景

<div id="usage-scene-graph-detail-content"></div>


（1）统计学家和医学改革家佛罗伦萨‧南丁格尔 (Florence Nightingale) 在克里米亚战争期间创造了南丁格尔玫瑰图，用以反映士兵的季节性死亡率。图中的每一片“花瓣”代表一个月里的士兵死亡量，其中粉红色表示因受伤过重而死亡的士兵，蓝色表示死于原本可避免的感染的士兵，黑色表示死于其他原因的士兵。右图包含<span style={{"color":"rgb(0, 0, 0)","backgroundColor":"rgb(255, 255, 255)"}}>1854年4月至1855年3月的数据，</span>左边的玫瑰图为1855年4月到1856年3月的数据。南丁格尔成功用这张图说服了上级——最多的士兵不是死在战场上，而是死在恶劣的医疗条件上，从而促进了军医院环境的改善。[[点击阅读更多]](http://blog.infographics.tw/2016/02/nightingale-rose-chart/)

![](//i.ibb.co/zVL4Q5d/1546505391951-d6740156-b926-4259-b4ac-a0cb72218cac.png)

（2）作为柱状图的替代。例如，DT财经（<span style={{"backgroundColor":"rgb(255, 255, 255)","color":"rgb(61, 70, 77)"}}>唐也钦, 2017</span>）统计了中国星巴克店铺数量最多的城市，以径向的方式排列。上海约有500多家星巴克，数量遥遥领先，北京有200多家，苏州、杭州、深圳、广州约有100多家，在南丁格尔图上与上海的差距显示非常大。[[点击访问原作品网页]](https://dtcj.com/topic/493)

<span style={{"backgroundColor":"rgb(255, 255, 255)"}}>![](//i.ibb.co/4pjKzqL/1548668885879-2cd8e92d-d510-42b3-9ce1-f6dddf135b70.png)</span>

（3）下图（<span style={{"backgroundColor":"rgb(255, 255, 255)","color":"rgb(38, 38, 38)"}}>Excel How To, 2016</span>）为facebook的用户画像。每一块扇形表示一种用户群体，例如，41%的用户每天登陆；30%的用户通过移动端登陆。而教育（education）、年龄（age）、收入（income）、性别（gender）部分则用了层叠的玫瑰图。例如，46%的用户为男性，54%的用户为女性。[[点击访问原作品网页]](https://www.excelhowto.com/nightingale-rose-diagram-excel-template/)


![](//i.ibb.co/949rQ3P/1546508943869-7e3319d9-5464-4619-9e46-8b63ce16e6ba.png)
